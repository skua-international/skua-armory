# Settings

CBA Settings, managed by the mod so mission makers have one less file to copy.

Note: The [PBOPREFIX](./$PBOPREFIX$) file in this addon should be kept as-is as CBA loads a static filepath.

`force` should not be used, generally, as it disallows in-mission editing, even as a logged-in admin or settings-whitelisted user.

Setting a value should be good enough for 99% of use cases.
